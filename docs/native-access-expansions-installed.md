# Installed NI “Expansions” inferred from `/Users/Shared`

Native Access “Expansions” don’t all install with the same on‑disk layout. The items visible in the screenshot map to **three distinct folder styles** under `/Users/Shared`:

- **Maschine expansion style**: contains `Groups/` + `Sounds/` (usually also `Projects/` + `Samples/`).
  - Examples in screenshot: `Anima Ascent`, `Aquarius Earth`
- **Kit expansion style**: contains `Kits/` + `Samples/` (often also `Effects/`).
  - Examples in screenshot: `80s New Wave`, `Afrobeats`
- **Preset pack style**: contains `Presets/` + `PAResources/` (often no `Samples/` at the top level).
  - Example in screenshot: `Ambient Atmospheres`

Using those signatures (and excluding Kontakt libraries that contain `Instruments/`), the following expansion‑like folders appear to be installed.

Notes:

- Names are taken from folder names under `/Users/Shared`.
- Items marked **(factory/base)** are libraries, not “Expansion packs”, but they match the same structural pattern.
